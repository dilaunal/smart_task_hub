# 🚀 Smart Task Hub

Modern, hızlı ve kullanıcı dostu bir görev yönetim uygulaması. 
Bu proje, bir yazılım geliştiricinin temel yetenekleri olan CRUD operasyonlarını, 
API yönetimini ve modern web teknolojilerini bir araya getirir.

## 🛠️ Teknolojiler
- **Frontend:** Next.js, React, Tailwind CSS (Vercel ile canlıda)
- **Backend:** Python, FastAPI (Render ile canlıda)
- **Veritabanı:** SQLAlchemy & SQLite
- **DevOps:** Git, GitHub, CI/CD süreçleri

## 💡 Öne Çıkan Özellikler
- **Tam CRUD Döngüsü:** Görev ekleme, listeleme, güncelleme ve silme.
- **Modern UI/UX:** Glassmorphism tasarımı ve responsive (mobil uyumlu) yapı.
- **Hata Yönetimi:** FastAPI HTTPException ile güvenli API yanıtları.
- **CORS Yapılandırması:** Farklı sunucular arası güvenli veri trafiği.
